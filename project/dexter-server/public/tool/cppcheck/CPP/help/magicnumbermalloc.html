Checker Code
  magicNumberMalloc
Description
  4(magic number) is used in malloc.
Severity
ETC
Detector / Bug Pattern
4(magic number) is used in malloc.
More Information
32-bit and 64-bit Linux they are different pointer size.
please use sizeof() in malloc instead of 4(magic number).
Bad Code 
void myfunction()
{
	size_t num=10;
	
	int **pointerArray1 = (int **) malloc(num*4);
}
Good Code 
void myfunction()
{
	size_t num=10;
	
	int **pointerArray2 = (int **) malloc(sizeof(int *));
}
