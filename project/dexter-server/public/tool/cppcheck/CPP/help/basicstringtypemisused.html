Description
  std::basic_string type misused. (C++98)
Severity
Critical
Detector / Bug Pattern
std::basic_string character type misused. Use only (char or wchar_t) types in basic_string (C++98).
More Information
std::basic_string character type misused.
(short|int|long|long long|size_t|float|double) is used in std::basic_sitring. Use only (char or wchar_t) types in basic_string (C++98).
Bad Code 
typedef std::basic_string < unsigned short, std::char_traits <unsigned short>, std::allocator< unsigned short> > ustring;
typedef std::basic_string < unsigned int, std::char_traits <unsigned int>, std::allocator< unsigned int> > uint;

void my_func(void)
{
	std::ustring s1;
	std::uint s2;
}
Good Code 
typedef std::basic_string < char, std::char_traits < char> , std::allocator< char> > uchar;
typedef std::basic_string < wchar_t, std::char_traits < wchar_t> , std::allocator < wchar_t> > uwchar_t;
	
void my_func(void)
{
	std::uchar s1;
	std::uwchar_t s2;
}
